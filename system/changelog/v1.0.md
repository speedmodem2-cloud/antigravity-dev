# AG 개발환경 v1.0 — 초기 구축

> 날짜: 2026-02-16

## 개요

AntiGravity 멀티 에이전트 개발환경 최초 구축.

## 포함 내용

- 에이전트 5종 정의 (아키텍트, 개발자, 리뷰어, 테스터, 문서화)
- MCP 서버 4종 (project-manager, token-monitor, image-bridge, gemini-bridge)
- 파이프라인 단계 전환 조건 (6단계: 스키마→배포)
- 매뉴얼 (코딩 규칙, 리뷰 체크리스트, 품질 게이트 4개, 토큰 최적화)
- 템플릿 (web-project.md, vite-config-template.ts, deploy.sh)
- TUI 대시보드, agent-factory, retrospect 도구
- 공유 스니펫/에셋 구조

## 첫 프로젝트

- workspace/portfolio (Alex Kira 포트폴리오)
