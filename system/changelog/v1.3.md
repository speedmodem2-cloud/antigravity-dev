### New: Document Split (AI/User)

- `system/manuals/` → AI-facing (English, compressed, keyword-centric)
- `system/docs/dev-guide.md` → User-facing (Korean, readable, consolidated)

### New: Phase Manager CLI (`tools/phase-manager/`)

- Commands: init, advance, complete, skip, status, reset

- Added: phase completion analysis (skipped phase detection)

- Output: comprehensive KPT report with all dimensions

- Added: doc split convention
