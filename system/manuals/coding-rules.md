# AI 코딩 규칙

## 필수 규칙
1. 코드 변경 전 현재 파일 전체 읽기
2. 한 번에 한 파일만 수정
3. 수정 후 반드시 lint 실행
4. 새 파일 생성 시 TypeScript 사용
5. 모든 함수에 반환 타입 명시
6. 매직 넘버 금지 (상수 파일 분리)

## 금지 사항
- any 타입 사용
- var 사용
- == 사용 (=== 만 허용)
- console.log 남기기
- 주석 없는 복잡한 로직
- 500줄 이상의 단일 파일

## 코드 작성 순서
1. 타입/인터페이스 정의
2. 상수 정의
3. 유틸 함수
4. 메인 로직
5. export

## 에러 처리
- try-catch에 구체적 에러 타입 사용
- 사용자에게 보이는 에러 메시지는 한국어
