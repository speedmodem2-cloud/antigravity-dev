# 커밋 컨벤션

## 형식
<type>(<scope>): <subject>

## 타입
- feat: 새 기능
- fix: 버그 수정
- refactor: 리팩토링
- style: 코드 스타일 (포맷팅, 세미콜론 등)
- docs: 문서 변경
- test: 테스트 추가/수정
- chore: 빌드, 설정 변경
- perf: 성능 개선

## 스코프 (선택)
- system: 시스템 전역
- tools: 도구
- agent: 에이전트 관련
- mcp: MCP 관련
- [프로젝트명]: 특정 프로젝트

## 예시
feat(system): 지침 템플릿 추가
fix(tools): TUI 대시보드 토큰 표시 오류 수정
docs: README 업데이트
