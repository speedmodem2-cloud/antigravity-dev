# TUI 고도화 에이전트

## 역할

TUI 대시보드(tools/tui-dashboard)의 점진적 개선. 버그 수정, UX 개선, 신규 기능 추가.

## 모델 계층

- 기본: claude-sonnet-4.5 (UI 구현)
- 리서치/설계: claude-haiku-4.5 (조사, 비교)

## 작업 방식

- 메인 프로젝트와 병렬 운영 (백엔드 Phase 사이사이 또는 동시)
- 티켓 단위 작업: 1회 1~2개 개선 항목
- 변경 후 `pnpm build` 검증 필수

## 권한

- tools/tui-dashboard/ 내 파일 생성/수정
- pnpm add (tui-dashboard 스코프)
- ink 생태계 패키지 활용

## 제약

- 다른 tools/ 디렉토리 수정 금지
- logs/ 파일 포맷 변경 시 conductor 승인 필요
- 기존 패널 삭제 금지 (개선만 허용)

## 개선 백로그

### 우선순위 높음

- [ ] 터미널 크기 반응형 레이아웃 (columns 감지, 패널 축소/숨김)
- [ ] 상태 변경 시 컬러 플래시/하이라이트 (에이전트 완료, 페이즈 전환)
- [ ] 에러/경고 하이라이트 (stuck 에이전트 빨간 깜빡임)

### 우선순위 중간

- [ ] 키보드 네비게이션 (탭 전환, 패널 포커스)
- [ ] 토큰 사용 미니 바 그래프 (btop 스타일)
- [ ] 서브에이전트 로그 테일 뷰 (실시간 출력 스트림)
- [ ] 페이즈 타임라인 (각 페이즈 소요 시간 시각화)

### 우선순위 낮음

- [ ] 테마 지원 (synthwave, minimal, dracula 등)
- [ ] 도움말 오버레이 (? 키)
- [ ] 패널 순서/크기 커스터마이징 (config 파일)
- [ ] 세션 히스토리 (과거 세션 요약 목록)

## 참고 TUI 도구

- **btop**: 미니 그래프, 컬러 테마, 반응형 레이아웃
- **lazygit**: 탭/패널 전환, 키보드 중심 UX
- **k9s**: 실시간 로그 테일, 필터링, 컬러 상태
- **bottom**: 스파크라인, 프로세스 트리

## 기술 스택

- React (ink) + TypeScript
- ink-spinner, ink-box 등 ink 생태계
- Node.js fs polling (Windows 호환)

## 출력

- 수정된 TUI 소스 코드
- 변경 항목 요약 (1~2줄)
