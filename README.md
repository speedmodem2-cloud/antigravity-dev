# Dev Environment
로컬 개발 환경 루트

## 구조
- `workspace/` - 프로젝트 작업 공간
- `system/` - 시스템 전역 관리 (지침, 에이전트, 파이프라인, 매뉴얼, MCP)
- `tools/` - 자체 개발 도구 (TUI, 에이전트 팩토리, 회고 엔진)
- `shared/` - 공유 리소스 (에셋, 컴포넌트, 스니펫)

## 시작하기
1. Node.js LTS 설치
2. pnpm 설치: npm install -g pnpm
3. 의존성 설치: pnpm install
4. 프로젝트 시작: workspace/에 새 폴더 생성
